# 📊 Data Cleaning & Visualization Dashboard

An automated, interactive web application built with Python and Streamlit to streamline the data preprocessing pipeline and generate exploratory data visualizations.

## 🚀 Overview
Preparing data for analysis or machine learning is often the most time-consuming part of any data project. This dashboard allows users to upload raw datasets, automatically handle missing values, filter outliers, and instantly generate interactive visualizations to understand their data distribution without writing any repetitive code.

## ✨ Key Features
* **Automated Data Cleaning:** One-click imputation for missing values (mean, median, mode) and duplicate removal.
* **Exploratory Data Analysis (EDA):** Quick statistical summaries and structural overviews of uploaded datasets.
* **Dynamic Visualizations:** Interactive scatter plots, bar charts, and histograms to identify trends and outliers.
* **Export Ready:** Download the cleaned dataset as a fresh CSV file ready for downstream machine learning tasks.

## 🛠️ Technology Stack
* **Language:** Python
* **Framework:** Streamlit
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn (or Plotly)
